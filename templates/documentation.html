Quran API Documentation

GET /quran/surahs

Returns a list of all surahs in the Quran.

Request

GET /quran/surahs

Response

Field Type Description
surahs array An array of objects representing each surah in the Quran.
surah_number integer The number of the surah.
name string The name of the surah.
english_name string The English name of the surah.
english_name_translation string The English translation of the name of the surah.
number_of_ayahs integer The number of ayahs in the surah.

GET /quran/surah/{surah_number}

Returns the details of a specific surah in the Quran.

Request

GET /quran/surah/{surah_number}

Response

Field Type Description
surah_number integer The number of the surah.
name string The name of the surah.
english_name string The English name of the surah.
english_name_translation string The English translation of the name of the surah.
number_of_ayahs integer The number of ayahs in the surah.
ayahs array An array of objects representing each ayah in the surah.

GET /quran/ayah_number/{ayah_number}

Returns the details of a specific ayah in the Quran.

Request

GET /quran/ayah_number/{ayah_number}

Response

Field Type Description
surah_number integer The number of the surah the verse is in
ayah_number_within_surah integer Aya number within the surah it is in
ayah_content string The content of the aya.
